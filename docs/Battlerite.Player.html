Class: Player

Battlerite.Player(playerData, reorganize)

Class Player, containing data about a single player and static methods used to retrieve Player data from the API.

Constructor

new Player(playerData, reorganize)

Constructor for the Match class.
Parameters:
Name Type Default Description
playerData Object data about the player
reorganize boolean true tells the constructor whether it has to reorganize the player data or not. if false, playerData is just copied inside the new match. If true, data is organized so that attributes are moved directly inside the Player Object. this allows to copy data between two different Player objects.
