Class: Team

Battlerite.Team(teamData, reorganize)

Class Team, containing data about a single team and static methods used to retrieve team data from the API.

Constructor

new Team(teamData, reorganize)

Constructor for the Team class.
Parameters:
Name Type Default Description
teamData Object data about the team
reorganize boolean true tells the constructor whether it has to reorganize the team data or not. if false, teamData is just copied inside the new team. If true, data is organized so that attributes are moved directly inside the Team Object. this allows to copy data between two different Team objects.
